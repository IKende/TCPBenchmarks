# tcp,websocket,webapi性能测试工具

### 国内地址

https://gitee.com/ikende/TCPBenchmarks

### linux下使用
```
sudo chmod 777 ./TCP-BenchmarkApp
sudo ./TCP-BenchmarkApp
//指定服务端口
sudo ./TCP-BenchmarkApp port=8088
```
## windows下使用
```
TCP-BenchmarkApp.exe
//指定端口
TCP-BenchmarkApp.exe port=8088
```
